# E-Commerce Backend API

A simple e-commerce backend application built using Spring Boot.

## Features

* Create products
* View all products
* View product by ID
* Update products
* Delete products
* Search products
* Upload product images
* Retrieve product images

## Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* H2 Database
* Maven

## API Endpoints

* `GET /api/product`
* `GET /api/product/{id}`
* `POST /api/product`
* `PUT /api/product/{id}`
* `DELETE /api/product/{id}`
* `GET /api/product/{id}/image`
